Download zip file from: https://github.com/netsphere-labs/mb_PetitLyricsPlugin
mb_PetitLyricsPlugin-master.zip


1. Unzip the contents and compile the source code with Visual Studio.
2. You will get mb_PetitLyricsPlugin.dll
3. If you don't know how to use Visual Studio, you can download the DLL file from here.
4. Download the DLL file and copy it into C:\Program Files (x86)\MusicBee\Plugins\
5. Change settings in Musicbee accordingly
	- Tab(2):  Activate Petit Lyrics 
	- Plugin:  Possibly you don't need to do anything here
